# QDialog - Scroll position is not the same after closing a maximized dialog

### What happened?

When you have a long page with a scrolling body, the scroll position after opening/closing a maximized Dialog changes. 

### What did you expect to happen?

It should stay at the exact same scroll position after opening/closing a dialog. 

### Reproduction URL

https://stackblitz.com/edit/quasarframework-rqbey1

### How to reproduce?

1. Click on a link in the left drawer to move to a section.
2. Open a dialog using a button.
3. Close the dialog
4. Check scroll position
