# quasar-v2.18.2

## New
* feat(ui): Albanian Quasar language pack (#17985)

## Changes
* fix(Dark): fix flash of light theme with SSR, fixes #17953 (#17965)
* fix(ui): fix __splitDate function: invalid call function 'pad' for date.year (#17907)
* fix(QEditor): consistent use of htmlTip in editor buttons #16933 (#17899)
* fix(QEditor): remove tabindex: -1 from remove link button, add aria-label to QEditor buttons (fix: partially #15675, #13275) (#17893)
* fix(ui): replace ariaLabel with aria-label (fix: #17988) (#17996)
* fix(ui): remove double border-top in no data scenario (fix #18006) (#18020)
* fix(ui): loosen getMinDate/getMaxDate types to allow spread usage (fix: #17858) (#18047)
* fix(QTable): support meta key for row selection (fix #18042) (#18043)
* fix(QInput): useMask wrongly consumes character (fix #17980) (#18018)
* fix(AddressbarColor): unsupported tag 'apple-mobile-web-app-status-bar-style' replaced by supported tag 'theme-color' for iOS/Safari (fix: #17900) (#17989)
