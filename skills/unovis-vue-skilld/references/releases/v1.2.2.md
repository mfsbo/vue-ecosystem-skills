# Release 1.2.2

This release of Unovis contains a few tweaks and fixes. See details below.

## What's new
* Component | BulletLegend: Added support for bullet shapes https://github.com/f5/unovis/pull/280
* Component | Axis: Setting label and tick text color via config https://github.com/f5/unovis/pull/268
* Container | XY: Smarter `preventEmptyDomain` handling https://github.com/f5/unovis/pull/270
* Utils | Text: Supporting `fontWeight` in UnovisText; Supporting `x` and `y` in `UnovisTextOptions` (it was supported only by `UnovisTextFrameOptions` before). https://github.com/f5/unovis/pull/274

## Fixes:
* Component | Graph: Autofit the layout if it was recalculated https://github.com/f5/unovis/pull/284
* Component | Nested Donut | Using `fill-opacity` for segment shading https://github.com/f5/unovis/pull/277

**Full Changelog**: https://github.com/f5/unovis/compare/1.2.1...1.2.2
