# v3.13.3

## VueUiSparkline

- Add optional pulse in line mode
- The pulse will always be disabled if users have prefers-reduced-motion

https://github.com/user-attachments/assets/c6108567-19a9-4c74-a5de-dfa8e06a6362

```js
const config = ref({
  style: {
    line: {
      pulse: {
        show: false,
        loop: true,
        color: '#FFFFFF',
        durationMs: 4000,
        easing: 'ease-in-out',
        radius: 4,
        trail: {
          show: true,
          length: 6,
          opacity: 0.75,
          strokeWidth: 3
        }
      }
    }
  }
})
```

Docs are up to date
