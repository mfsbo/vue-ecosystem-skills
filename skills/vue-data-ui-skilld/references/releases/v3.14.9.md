# v3.14.9

## VueUiXy

Add new emits related to the zoom component:

- `zoomStart`, `zoomEnd`

```html
<VueUiXy :dataset="dataset" :config="config" @zoomEnd="handleZoom" />
```

```js
function handleZoom({ index, isZoom }) {
  console.log(index, isZoom);
}
```

- `zoomReset`, emitted when the zoom is reset from the reset button
