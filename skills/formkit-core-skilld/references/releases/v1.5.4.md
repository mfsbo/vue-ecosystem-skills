# 1.5.4

###  New features

- Floating Labels plugin now supports `prefix` content when calculating label position. (#878)
- Updates the `@formkit/cli` package to allow generating of published themes locally in your projects. See the new documentation [here](/guides/create-a-tailwind-theme#installing-a-published-theme). (#1170)

###  Bug fixes

- Fixes issue where `multi-step` would not allow schema overrides of `tabs` or `tab` sections. (1110)
