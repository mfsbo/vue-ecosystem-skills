# v2.4.0

### &nbsp;&nbsp;&nbsp; Breaking Changes

- **VisuallyHidden**: Remove curly brackets in prop name format &nbsp;-&nbsp; by @kricsleo in https://github.com/unovue/reka-ui/issues/2017 <samp>(e0acb)</samp>

### &nbsp;&nbsp;&nbsp; Features

- **Combobox**: Add `openOnFocus` and `openOnClick` props &nbsp;-&nbsp; by @markjaniczak in https://github.com/unovue/reka-ui/issues/2050 <samp>(64ea5)</samp>
- **NumberField**: Support `readonly` for the `NumberField` &nbsp;-&nbsp; by @kricsleo in https://github.com/unovue/reka-ui/issues/2016 <samp>(987f4)</samp>
- **Popover**: Add programmatic close functionality to popover root &nbsp;-&nbsp; by @JosephAnson in https://github.com/unovue/reka-ui/issues/1956 <samp>(3a399)</samp>
- **Popper**: Add flip options props &nbsp;-&nbsp; by @aguingand in https://github.com/unovue/reka-ui/issues/1684 <samp>(678eb)</samp>
- **ScrollArea**: Add data-state support for ScrollAreaScrollbar scroll type &nbsp;-&nbsp; by @vowdemon in https://github.com/unovue/reka-ui/issues/1980 <samp>(c7dfe)</samp>
- **Splitter**: Add CSP nonce support &nbsp;-&nbsp; by @Y0me in https://github.com/unovue/reka-ui/issues/2038 <samp>(afbde)</samp>
- **calendar**: Add `getWeekNumber` utility function &nbsp;-&nbsp; by @J-Michalek in https://github.com/unovue/reka-ui/issues/1959 <samp>(09202)</samp>
