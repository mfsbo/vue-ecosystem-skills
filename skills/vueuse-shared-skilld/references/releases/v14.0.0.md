# v14.0.0

### &nbsp;&nbsp;&nbsp; Breaking Changes

- Migrate to `tsdown`, move dist files &nbsp;-&nbsp; by @9romise and **Anthony Fu** in https://github.com/vueuse/vueuse/issues/5004 <samp>(a2e2c)</samp>
- Deprecate alias exports in favor of original function names &nbsp;-&nbsp; by @9romise in https://github.com/vueuse/vueuse/issues/5009 <samp>(e5f74)</samp>
- Requires Vue 3.5 &nbsp;-&nbsp; by @antfu <samp>(fe458)</samp>
- **components**: Refactor components and make them consistent &nbsp;-&nbsp; by @ilyaliao in https://github.com/vueuse/vueuse/issues/4912 <samp>(8c521)</samp>
- **computedAsync**: Default to `flush: sync` &nbsp;-&nbsp; by @ferferga in https://github.com/vueuse/vueuse/issues/4752 <samp>(573bf)</samp>
- **createSharedComposable**: Return only the sharedComposable on client side &nbsp;-&nbsp; by @9romise in https://github.com/vueuse/vueuse/issues/4997 <samp>(73aad)</samp>
- **firebase**: Upgrade to firebase v12 &nbsp;-&nbsp; by @antfu <samp>(8cb0b)</samp>
- **nuxt**: Use Nuxt v4 kit &nbsp;-&nbsp; by @antfu <samp>(070fc)</samp>
- **shared**: Drop some deprecated apis &nbsp;-&nbsp; by @9romise in https://github.com/vueuse/vueuse/issues/5010 <samp>(971b2)</samp>
- **useClipboard**: Use `readonly()` instead of type assertion `Computed` &nbsp;-&nbsp; by @9romise in https://github.com/vueuse/vueuse/issues/5081 <samp>(67812)</samp>
- **useSwipe**: Remove isPassiveEventSupported &nbsp;-&nbsp; by @pkc918 in https://github.com/vueuse/vueuse/issues/4886 <samp>(620a9)</samp>
- **useThrottleFn**: Align with traditional throttle behavior &nbsp;-&nbsp; by @IceMooncake and @ilyaliao in https://github.com/vueuse/vueuse/issues/4923 <samp>(cefd2)</samp>
