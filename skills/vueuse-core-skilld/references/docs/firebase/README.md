# @vueuse/firebase



Add-on of VueUse, enables the real-time bindings for Firebase.

>  This package only work with Firebase 9 or above with the modular style. For legacy versions, use `@vueuse/firebase@8` instead.

>  Give a try to VueFire if you are looking for a more complete solution!

## Install

```bash
npm i @vueuse/firebase firebase
```

## Functions

`@vueuse/firebase` provides the following functions




- `useAuth` — reactive Firebase Auth binding
- `useFirestore` — reactive Firestore binding
- `useRTDB` — reactive Firebase Realtime Database binding
