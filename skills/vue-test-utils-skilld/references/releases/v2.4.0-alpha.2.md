# v2.4.0-alpha.2

Note: It has no API related breaking changes, but some types might be impacted - the full extent of https://github.com/vuejs/test-utils/pull/2026 is still being investigated, but at this point it's unlikely you'll need to change any code. In general, type safety should be improved across the board.
