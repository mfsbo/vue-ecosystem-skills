# Beta Release Checklist

The following is an in progress list of remaining tasks before we release beta. It will be updated daily; so watch for changes.

Discord, https://community.vuetifyjs.com/ and join channel **#v3-discussion**.

- [x] `v-autocomplete`
- [x] `v-carousel`
- [x] `v-combobox`
- [x] `v-tabs`
- [x] `v-snackbar`
- [x] enable locale download and compilation in docs
- [x] update preset usage information
- [x] triage pending v3.0.0-alpha issues
- [x] review pending v3.0.0-alpha PRs
- [x] update issue generator

**Known Bugs**
- `v-app-bar` missing scrolling-techniques
- `v-text-field` with default density and outlined variant misaligned
- `v-list-item` (prepend/append slots)
- `v-system-bar` missing layout support
